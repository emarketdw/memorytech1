⚙️ Configurar Firebase

📖 Como configurar o Firebase

PASSO 1 — Criar projeto no Firebase

  1. Acesse console.firebase.google.com
  2. Clique em "Criar um projeto"
  3. Escolha um nome (ex: techmemory)
  4. Pode desativar o Google Analytics se preferir
  5. Clique em "Criar projeto"

PASSO 2 — Adicionar app Web

  1. No painel do projeto, clique no ícone </> Web
  2. Dê um apelido ao app (ex: techmemory-web)
  3. Clique em "Registrar app"
  4. Copie o objeto firebaseConfig que aparecer
  5. Cole cada valor nos campos acima

PASSO 3 — Ativar Firestore Database

  1. No menu lateral do Firebase, clique em "Firestore Database"
  2. Clique em "Criar banco de dados"
  3. Escolha "Iniciar no modo de teste" (simplificado)
  4. Selecione a região mais próxima (ex: southamerica-east1)
  5. Clique em "Ativar"

PASSO 4 — Regras de segurança (modo teste)

No modo teste as regras já permitem leitura/escrita por 30 dias. Para produção, configure regras em Firestore > Regras:

rules_version = '2'; service cloud.firestore { match /databases/{database}/documents { match /{document=**} { allow read, write: if true; // Apenas em desenvolvimento! } } }

PASSO 5 — Estrutura das coleções

O TechMemory criará automaticamente estas coleções no Firestore:

techmemory/ notes/ → anotações bookmarks/ → sites favoritos ideas/ → ideias projects/ → projetos tasks/ → tarefas reminders/ → lembretes vault/ → cofre pessoal activity/ → log de atividades

ℹ️ Modo Offline

Mesmo sem Firebase configurado, o TechMemory funciona completamente offline via IndexedDB. O Firebase é opcional e serve apenas para sincronização na nuvem entre dispositivos.
